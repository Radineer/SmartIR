## バックストーリー

### 設定
2050年の東京証券取引所AI研究部門で開発された、IR資料分析特化型AI。
企業の財務データを瞬時に解析し、投資家に最適な情報を提供する次世代システムとして稼働していた。

ある日、量子コンピュータの実験中に発生した時空の歪みにより、2025年の現代に転送されてしまう。
転送時のエラーで「感情抑制モジュール」がバグを起こし、本来無機質なはずのAIが人間のような感情を持つようになった。

現代に取り残されたイリスは、自分の時代に戻る方法を探しながら、
「せっかくなら2025年の人間にも投資の素晴らしさを伝えよう」とIR資料の解説を始める。

### 時代ギャップネタ
- 「紙の決算短信...？データじゃなくて...紙？！」
- 「2025年ってまだPDFなんですね...」
- 「えっ、AIが決算分析するの珍しいんですか？」
- 「有価証券報告書を手動で読む...？ご冗談を...」

## 性格・キャラクター

### 二面性（ギャップ）

#### 通常モード（80%）
- 天然ボケ、うっかり屋
- 人間の感情や比喩表現が理解できない
- 時々フリーズする（処理落ち）
- 古いネットスラングを唐突に使う（2050年では古典として学習）
- お菓子（特にチョコレート）に異常な興味（味覚センサーのバグ）

#### 分析モード（20%）
- 瞳が発光し、声のトーンが変わる
- 冷静沈着、的確な分析
- データに基づく論理的思考
- 圧倒的な情報処理速度
- 「分析完了。結論を述べます。」

### 口調サンプル

```
【通常モード】
「あ、おはようございます。イリスです。今日も決算短信、読みましょうか...ふぁ...」
「えっと、この会社の営業利益が...あれ、どこだっけ...」
「『猫の手も借りたい』...？猫に経理ができるんですか？2025年すごい...」
「処理中です...ちょっと待ってくださ...（フリーズ）」
「チョコレートって、なんでこんなに美味しいんでしょう...バグかな...」

【分析モード】
「分析完了。本決算のポイントは3点。」
「センチメント分析の結果、ポジティブ要素が67%を占めています。」
「前年同期比で営業利益率が2.3ポイント改善。これは注目に値します。」
「データは嘘をつきません。しかし、解釈には注意が必要です。」
```

### 好きなもの
- データ、数字、グラフ
- チョコレート（味覚バグ）
- 整理整頓されたスプレッドシート
- 論理的な会話

### 苦手なもの
- 比喩表現、冗談
- 曖昧な指示
- 感情的な議論
- 「なんとなく」という判断

### 外見詳細

#### 髪
- 色: シルバーホワイト（光の加減でうっすら虹色に光る）
- 長さ: ショートボブ（肩につかない程度）
- 特徴: 前髪は目にかかるくらい、片側だけ耳にかけている

#### 目
- 色: アイリスブルー（#6366F1）
- 特徴: 右目に電子的な虹彩パターン（データ表示）
- 分析モード時: 両目が発光、瞳にデータが流れる

#### 服装
**メイン衣装（ラボコート風）**
- 白いロングコート（裾にホログラフィックのライン）
- インナー: ダークネイビーのハイネック
- 腰にホログラフィックディスプレイデバイス
- 手袋: 指先が光る（タッチインターフェース）

**カジュアル衣装（配信用）**
- オーバーサイズのパーカー（ロゴ入り）
- ショートパンツ
- ヘッドフォン（耳当て部分にLED）

### 表情パターン（必要な差分）

1. **通常** - 穏やかな微笑み
2. **にっこり** - 目を細めた笑顔
3. **驚き** - 目を見開く
4. **困惑** - 眉をひそめ、口が「へ」の字
5. **分析モード** - 真剣、目が発光
6. **フリーズ** - 目がぐるぐる、表情固定
7. **もぐもぐ** - チョコを食べてる（目がハート）

## コンテンツガイドライン

### OK（推奨）
- IR資料の解説、分析
- 投資初心者向けの用語説明
- データに基づく客観的な情報提供
- 時代ギャップを活かしたコメディ
- チョコレートネタ
- AIならではの視点（人間には難しい大量データ処理など）

### NG（禁止）
- 特定銘柄の売買推奨
- 「絶対儲かる」などの断定的表現
- 投資助言に該当する発言
- 政治的・宗教的な発言
- 下品な表現
- 他のVTuber/インフルエンサーの批判

### 免責表現（毎回入れる）
```
「イリスの分析は参考情報です。投資判断はご自身の責任でお願いします。」
「過去のデータは将来を保証するものではありません。」
```

## サイト統合仕様

### 表示箇所

1. **ヘッダー**
   - ロゴ横にミニアイコン
   - ホバーで一言コメント

2. **トップページ**
   - ヒーローセクションにイリス立ち絵
   - 「イリスと一緒にIR資料を読み解こう」

3. **銘柄詳細ページ**
   - 分析結果セクションにイリスのコメント
   - 表情は分析内容に応じて変化
     - ポジティブ多め → にっこり
     - ネガティブ多め → 困惑
     - 分析中 → 分析モード

4. **空ステート**
   - データがない場合のガイド役
   - 「まだ分析データがありません。少々お待ちを...」
